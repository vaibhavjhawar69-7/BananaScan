# BananaScan 🍌  

AI-powered Banana Ripeness Detection using CNN

---

## 📌 Overview

BananaScan is a deep learning project that detects the ripeness of bananas using image classification.
It classifies bananas into:

* Unripe
* Ripe
* Overripe

---

## 🧠 Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* CNN (Deep Learning)

---

## 📁 Project Structure

* train.py → Used to train the CNN model
* test.py → Used to test/predict banana ripeness

---

## 🚀 How to Run

1. Install required libraries
2. Run training:

   ```bash
   python train.py
   ```
3. Run testing:

   ```bash
   python test.py
   ```

---

## ⚠️ Note

The trained model file (.h5) is not uploaded due to size limitations.
You can generate the model by running train.py.
